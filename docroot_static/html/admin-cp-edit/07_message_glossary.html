用語説明

すべてのキャンペーンに表示される項目

日付 キャンペーンの開催期間のみ表示
参加者数(全体) キャンペーン全体の参加者数（STEP1を完了したユーザ数）
参加者数(PCのみ) PCより参加登録したキャンペーンの参加者数
参加者数(モバイルのみ) モバイルより参加登録したキャンペーンの参加者数
参加者数(新規会員のみ) 当キャンペーンでブランドに新規登録したの参加者数
参加者数(既存会員のみ) 当キャンペーン以前にブランドに登録済みの参加者数

2016年01月01日以降に開催されたキャンペーンのみ表示される項目

キャンペーントップPV(全体) キャンペーントップページ（/campaigns/xxxx）のページビュー数（Google アナリティクスより取得）
キャンペーントップPV(PCのみ) PC経由でのキャンペーントップページ（/campaigns/xxxx）のページビュー数（Google アナリティクスより取得）
キャンペーントップPV(モバイルのみ) モバイル経由でのキャンペーントップページ（/campaigns/xxxx）のページビュー数（Google アナリティクスより取得）
キャンペーントップUU(全体) キャンペーントップページ（/campaigns/xxxx）のユニークユーザ数（Google アナリティクスより取得）

2016年01月01日以降に開催されたキャンペーン　かつ　LPを設定しているキャンペーン　のみ表示される項目

キャンペーンLPPV(全体) キャンペーンランディングページ（/page/xxxx）のページビュー数（Google アナリティクスより取得）
キャンペーンLPPV(PCのみ) PC経由でのキャンペーンランディングページ（/page/xxxx）のページビュー数（Google アナリティクスより取得）
キャンペーンLPPV(モバイルのみ) モバイル経由でのキャンペーンランディングページ（/page/xxxx）のページビュー数（Google アナリティクスより取得）
キャンペーンLPUU(全体) キャンペーンランディングページ（/campaigns/xxxx）のユニークユーザ数（Google アナリティクスより取得）

FBいいね！モジュールを使用している場合に表示される項目

新規Facebookいいね！数(アカウント名) キャンペーンのステップで、いいね！ボタンが押下された回数
既存Facebookいいね！数(アカウント名) キャンペーンのステップで、すでにいいね！済みであった回数

TWフォローモジュールを使用している場合に表示される項目

新規Twitterフォロー数(アカウント名) キャンペーンのステップで、フォローボタンが押下された回数
既存Twitterフォロー数(アカウント名) キャンペーンのステップで、すでにフォロー済みであった回数

IGフォローモジュールを使用している場合に表示される項目

新規Instagramフォロー数(アカウント名) キャンペーンのステップで、フォローボタンが押下された回数
既存Instagramフォロー数(アカウント名) キャンペーンのステップで、すでにフォロー済みであった回数

YTチャンネル登録モジュールを使用している場合に表示される項目

新規チャンネル登録数(アカウント名) キャンペーンのステップで、チャンネル登録ボタンが押下された回数
既存チャンネル登録数(アカウント名) キャンペーンのステップで、すでにチャンネル登録済みであった回数
